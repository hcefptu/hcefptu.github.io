## 2. Code Logic

Ở mã này ta sẽ chú ý đến đoạn này:

```
v6 = 16;
qmemcpy(v7, " 0", sizeof(v7));
  sub_4011B9(aInputName);
  scanf("%s", v8);
  v3 = 0;
  for ( i = 0; v3 < (int)strlen(v8); ++i )
  {
    if ( i >= 3 )
      i = 0;
    sprintf(Buffer, "%s%02X", Buffer, v8[v3++] ^ v7[i - 1]);
  }
  memset(v8, 0, sizeof(v8));
  sub_4011B9(aInputSerial);
  scanf("%s", v8);
  if ( !strcmp(v8, Buffer) )
    sub_4011B9(aCorrect);
  else
    sub_4011B9(aWrong);
  return 0;

```
v6 có giá trị biến là 16

Lệnh **qmemcpy(v7, " 0", sizeof(v7));** sẽ copy vào biến v7 là giá trị “ 0”,

Các giá trị “16”, “ “ và “0” lần lượt có mã hex là **0x10, 0x20 và 0x30**

Đoạn tiếp theo:

  ```
  
  sub_4011B9(aInputName);
  scanf("%s", v8);
  
  ```

sẽ là nhận 1 giá trị từ người dùng và gán vào biến v8.
Phần quan trọng nhất là đoạn logic này :

> **sprintf(Buffer, "%s%02X", Buffer, v8[v3++] ^ v7[i - 1]);**

sprintf là hàm sẽ truyền tham số vào 1 biến theo 1 format định sẵn.
> **Syntax:** 'int sprintf(char *str, const char *format, ...);'

Đại loạn như nào truyền giá trị abc vào biến str với format có thể là %s, %d hoặc %c.
Ở phần đầu của code thì biến buffer được set với giá trị là 0

![images](assets/Reversing.Kr/keyGen/8.png)
_Hình 8: Giá trị của Buffer_

Quay lại với hàm sprintf ở trên ta sẽ thấy là sẽ truyền giá trị vào biến Buffer.
Với format là %s là string và %02X là truyền vào giá trị HEX có tối đa độ rộng là 2 kí tự. Ví dụ như 0x36.
2 tham số cuối là buffer và v8[v3++] ^ v7[i-1].

Logic của chỗ này là nó sẽ XOR từng giá trị của chuỗi mà ta nhập vào với 3 giá trị ban đầu của chương trình là 10, 20 và 30.

## 3. Decode Serial Number

Ở đây thì có vẻ như đoạn mã serial ban đầu là giá trị sau khi đã được XOR.

**5B134977135E7D13**

Với phép XOR ta sẽ có cơ chế là nếu: 

> *A ^ B = C thì C ^ B = A.*

Thế nên ta sẽ thử XOR ngược lại với 3 giá trị là 0x10, 0x20 và 0x30 xem thử sẽ ra gì.

```

5B ^ 0x10 = 4b
13 ^ 0x20 = 33
49 ^ 0x30 = 79
77 ^ 0x10 = 67
13 ^ 0x20 = 33
5E ^ 0x30 = 6e
7D ^ 0X10 = 6d
13 ^ 0x20 = 33

```

Chuỗi cuối cùng là:

**4b337967336e6d33**

Sau khi tra bảng ASCII thì ta có kết quả là : **K3yg3nm3**
